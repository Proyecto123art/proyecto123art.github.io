Descripción del Proyecto

Este proyecto, titulado "Prototipo de Red de Monitoreo y Alerta de Factores Climatológicos para el Servicio Nacional de Hidrografía Naval", está enfocado en mejorar la seguridad en la navegación en cuerpos de agua de Bolivia, especialmente en el Lago Titicaca. Mediante el uso de boyas meteorológicas equipadas con sensores, se recopilan datos de temperatura, presión atmosférica, humedad, y velocidad del viento. Estos datos se transmiten a una base de datos central a través de tecnología LoRa, donde se procesan para generar alertas tempranas en caso de condiciones climáticas adversas.

Antecedentes

El monitoreo de los cuerpos de agua ha sido una preocupación desde tiempos antiguos. Las primeras mediciones con boyas datan de los experimentos de Leonardo da Vinci. En la actualidad, el Servicio Nacional de Hidrografía Naval en Bolivia no cuenta con un sistema en tiempo real para monitorear factores climáticos y proporcionar alertas tempranas. La implementación de un sistema de boyas meteorológicas en el Lago Titicaca busca cubrir esta necesidad crucial para la seguridad de las embarcaciones.

Planteamiento del Problema

El principal problema es la falta de un sistema de monitoreo climatológico en tiempo real en Bolivia, lo que compromete la seguridad en la navegación. Sin un sistema confiable de alerta temprana, las embarcaciones están expuestas a condiciones meteorológicas peligrosas, como tormentas repentinas. El proyecto se pregunta: ¿cómo se podría mejorar la recolección y alerta de factores climatológicos que ayuden al Servicio Nacional de Hidrografía Naval?

Justificación

Este sistema es técnicamente relevante porque ofrece datos en tiempo real necesarios para planificar y gestionar recursos naturales, infraestructura y reducir riesgos para las comunidades cercanas. También es relevante desde una perspectiva académica, ya que aplica conocimientos de programación de microcontroladores, electrónica, y bases de datos, promoviendo una transferencia de conocimientos de la universidad hacia aplicaciones prácticas.

Objetivos del Proyecto

Objetivo General

Diseñar y desarrollar un prototipo de red de monitoreo y alerta de factores climatológicos para el Servicio Nacional de Hidrografía Naval, utilizando tecnología LoRa para la transmisión de datos en tiempo real.

Objetivos Específicos

  • Identificar los parámetros climatológicos necesarios para el monitoreo.
  • Diseñar las estructuras de las boyas meteorológicas.
  • Desarrollar una interfaz de usuario para notificaciones y alertas de riesgo.
  • Procesar y analizar los datos recopilados por las boyas.

Metodología

La metodología es de tipo aplicada y cuantitativa, ya que el proyecto se basa en la recolección y análisis de datos de sensores. Las boyas recopilan datos en tiempo real y los transmiten a través de una red LoRa a un servidor central donde se procesan para la emisión de alertas y análisis climático.

Alcance del Proyecto

  • Monitoreo de condiciones climatológicas en el Estrecho de Tiquina en el Lago Titicaca.
  • Implementación de una interfaz gráfica en tiempo real para la visualización y emisión de alertas.
  • Prototipado de dos boyas con sensores avanzados adaptables para otros cuerpos de agua en Bolivia.

Arquitectura y Componentes del Sistema

El proyecto utiliza tecnología LoRa para la comunicación de larga distancia y baja potencia entre las boyas y el servidor central. Cada boya incluye sensores para medir temperatura, humedad, presión y otros factores. La estructura del sistema incluye:

  • Microcontrolador TTGO T-Beam V1.1 para conectar las boyas a la red LoRa.
  • Paneles solares para la autosuficiencia energética de las boyas.
  • Sensores de humedad, temperatura, y viento.

Recursos y Presupuesto

El costo estimado del proyecto es de 4,642.00 bolivianos, incluyendo la adquisición de materiales como microcontroladores, sensores y paneles solares. Se han realizado cotizaciones en tiendas de La Paz especializadas en electrónica.

Cronograma de Actividades

El proyecto tiene una duración de tres meses, en los cuales se realizarán actividades de diseño, construcción de las boyas, implementación de la interfaz y pruebas en campo.
